







# RabbitMQ
- **message_queue** 消息队列的基本用法  
- **publish_subsribe** 发布-订阅模式的用法
- **rpc_queue** 远程进程调用的用法

##### Shell中简单命令
- `sudo rabbitmqctl list_queues` 显示队列信息 
- `sudo rabbitmqctl list_exchanges` 显示exchanges  
- `sudo rabbitmqctl list_bindings` 显示bindings
